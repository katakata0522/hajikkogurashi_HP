# ミニゲーム企画書：限界！ステルスサボタージュ（仮）

## 1. コンセプト・概要
* **ジャンル**: チキンレース・バカゲー
* **ターゲット**: 日常のストレスを発散したい人、クスッと笑えるネタゲーが好きな人
* **コンセプト**: 上司が後ろを向いている隙に、画面を長押しして全力でスマホゲーム（サボり）をする！上司が振り返った瞬間に指を離して「仕事してるフリ」をしなければ即クビ（ゲームオーバー）。

## 2. ゲームのルールと操作方法
* **基本操作**: 画面の**長押し（ホールド）**と**離す（リリース）**のみ。
* **プレイルーム**:
  1. **サボる（長押し）**: 画面を長押ししている間、プレイヤーはサボり（ゲームなど）を開始し、スコアが猛スピードで加算されていきます。
  2. **仕事する（離す）**: 指を離すと、慌ててPCに向かい「仕事してるフリ」をします。スコア加算は止まります。
  3. **上司の監視**: 
     * 画面奥にいる上司は、普段は後ろを向いて仕事をしています。
     * ランダムなタイミングで「！」マーク（警戒状態）が出た後、こちらを振り向きます（監視状態）。
     * 上司が振り向いている（監視状態）の時に「サボる（長押し）」をしていると、即ゲームオーバーです。
  4. **難易度の上昇**:
     * スコアが上がるにつれて、上司が「！」を出してから振り向くまでの時間が極端に短くなったり、フェイント（振り向くフリをして振り向かない）をかけてきたりします。

## 3. 画面構成（UI/UX）
* **画面レイアウト**: 縦横どちらでもOK。オフィスを俯瞰（手前に自分、奥に上司）する構図。
* **デザインテイスト**: コミック調（アメコミ風のポップな配色、または日本のギャグ漫画風）。太い枠線や集中線などを使うとバカゲー感が出ます。
* **キャラクター**:
  * **プレイヤー**: 後ろ姿。長押し中は「スマホをいじってエフェクトが出る」、離すと「カタカタとPCを打つ」。
  * **上司**: 奥のデスク。後ろ姿から、振り向くと「鬼の形相（怒りマーク）」。
* **UI要素**:
  * 画面上部：現在のサボりスコア（例：`サボり度: 15300`）。
  * 画面中央：上司の吹き出し（「！」や「ジロッ」）。

## 4. 演出・サウンド
* **サウンド**:
  * サボり中：チャリンチャリン（スコアが増える音）や、ピコピコ（ゲーム音）。
  * 仕事中：カタカタカタ…（キーボードを叩く音）。
  * 上司の警戒：ピコーン！（警告音）。
  * ゲームオーバー：ドカーン！（雷の音や「キサマァァ！」という怒号）。
* **視覚効果**:
  * サボっている時は背景がフィーバー（集中線など）になる。

## 5. 拡散（バズ）の仕組み
* **X（Twitter）シェアテキスト案**:
  * 「上司の目を盗んで【35,000】サボりました。バレてクビになりました。」 ＋ ゲームURL ＋ #限界ステルスサボタージュ #はじっこぐらし
  * 「【神業】上司のフェイントを見切り、【100,000】サボり達成！ 称号：プロニート」

## 6. 技術スタック・実装方針
* **言語**: HTML5 Canvas, Vanilla JavaScript
* **実装のポイント**: 
  * マウス/タッチの `down` と `up` イベントで状態を管理。
  * `requestAnimationFrame` を使ってスコアの加算と上司の状態（タイマー）を制御。
* **ディレクトリ構成予定**:
  ```text
  stealth-slacker/
  ├── index.html       (ゲーム画面)
  ├── style.css        (UI・コミック風スタイリング)
  └── script.js        (ゲームロジック・上司のAI)
  ```
